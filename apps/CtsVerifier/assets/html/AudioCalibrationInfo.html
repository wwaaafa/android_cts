This panel allows you to set levels for loopback signals

The process is as follows:

  1. Select the input and output devices on which the loopback is executing.
  2. Click "Left" to play a sine wave on the left channel of the selected output device.
  3. Use the volume control on the DUT and/or external peripheral to set an appropriate signal level.
  4. Repeat the process for the right channel.
